# anyraces

This is a set of scripts that function as a library to scrape motorsport schedules and a webserver to re-host the data.

Currently it supports the following series:
- NASCAR Cup Series
- NASCAR O'Reilly Auto Parts Series
- NASCAR Craftsman Truck Series
- NASCAR Whelen Modified Tour
- ARCA Menards Series
- NASCAR Pinty's Series
- NTT IndyCar Series
- Indy NXT Series
- Formula One
- IMSA WeatherTech SportsCar Championship
- IMSA Michelin Pilot Challenge

Additional data may be added by adding CSV files to the /data directory.
